







# 重启

* `reboot` 
* `shutdown -r now` 立刻重启
* `shutdown -r 10` 过10分钟自动重启
* `shutdown -r 20:35` 在时间为20:35时候重启

如果是通过 shutdown 命令设置重启的话，可以用 `shutdown -c` 命令取消重启 







# 关机
   
* `poweroff` 立刻关机 
* `shutdown -h now` 立刻关机
* `shutdown -h 10` 10分钟后自动关机 

如果是通过 shutdown 命令设置关机的话，可以用 `shutdown -c` 命令取消关机
